# 🧫 MathSlime [TO BE UPDATED]

A lightweight, ad-free math game for kids (4+). Solve arithmetic problems to pop falling slime blobs before they reach the bottom. Single HTML file, no frameworks, no build step.

**Play now:** [tristanth03.github.io/math_slime](https://tristanth03.github.io/math_slime/)

## Design Pillars

Simple, lightweight, free, honest. A 4-year-old should figure it out in 10 seconds without help.
