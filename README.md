# MathSlime

A lightweight, ad-free math game for kids ages 4-10. Solve mental arithmetic problems to pop approaching slime blobs. Single HTML file, no frameworks, no build step, runs offline.

## Quick Start

Open `index.html` in a browser. Sign in with Google to save progress and appear on the leaderboard, or play as a guest.

## Features

- 6 core levels (addition, subtraction, multiplication, division)
- Multiple choice (levels 1-2) or numpad input (levels 3+)
- Leaderboard with three categories: High Score, Fastest Clear, Most Levels
- Full offline support (after first load)
- No ads, tracking, or in-app purchases
- Mobile-first design

## Design Pillars

Simple, lightweight, free, honest. A 4-year-old should figure it out in 10 seconds without help.

## Tech Stack

- Pure HTML/CSS/JavaScript (single file)
- Firebase Auth (Google Sign-In) + Firestore for persistent progress
- No build step, no dependencies beyond CDN scripts
